# Sonhos de Ana

Landing page de alto padrão para a marca **Sonhos de Ana**, focada em design unissex e conforto atemporal.

## 🚀 Estrutura do Projeto

O projeto segue uma organização profissional de arquivos estáticos:

```text
sa/
├── assets/           # Ativos globais do site
│   ├── css/          # Folhas de estilo (opcional/futuro)
│   ├── js/           # Scripts (opcional/futuro)
│   └── img/          # Imagens e ícones da marca
├── .gitignore        # Arquivos ignorados pelo Git
├── index.html        # Página principal (Single Page)
├── styleguidesa.html # Guia de estilos e identidade visual
└── README.md         # Documentação do projeto
```

## ✨ Funcionalidades

- **Slider Editorial Estilo Instagram:** Navegação infinita e intuitiva de produtos.
- **Scroll Reveal:** Animações sutis ao rolar a página para uma experiência premium.
- **Responsividade Total:** Adaptado para mobile, tablets e desktops.
- **Integração Social:** Botões diretos para WhatsApp e Instagram.

## 🛠️ Tecnologias

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Phosphor Icons
- Google Fonts (Inter & Playfair Display)
